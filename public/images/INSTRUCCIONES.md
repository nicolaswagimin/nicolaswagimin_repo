# Cómo agregar las imágenes

Las imágenes deben estar físicamente en esta carpeta como archivos, NO en el archivo .gitkeep.

## Pasos:

1. Copia o mueve las imágenes a esta carpeta:
   IMG_0273.jpg
IMG_0276.jpg

2. Verifica que los archivos estén aquí:
   ```bash
   ls -la public/images/*.jpg
   ```

3. Las imágenes se mostrarán automáticamente en el Hero Section.

## Nota:
- El archivo `.gitkeep` es solo para mantener la carpeta en Git
- Las imágenes reales deben ser archivos físicos en esta carpeta
